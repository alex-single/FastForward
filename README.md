# FastForward

A [Steamodded](https://github.com/Steamopollys/Steamodded) QOL mod for Balatro that adds adjustable game speeds to the UI and a function that allows you to speedup the game by pressing the space bar.


## Features
- Adds a speed button to the HUD that cycles through: **0.5x → 1x → 2x → 4x → 8x**
- Hold **Space** to temporarily jump to 35x speed, releasing restores your previous speed.

## Requirements
- [Steamodded](https://github.com/Steamopollys/Steamodded) >= 1.0.0~BETA-1221a

## Installation
1. Click the green **Code** button on this page and select **Download ZIP**
2. Extract the ZIP anywhere (e.g. your Downloads folder)
3. Open the extracted folder and double-click **install.bat**

The script copies the mod files to `%APPDATA%\Balatro\Mods\FastForward\` automatically.

**Manual install:** Copy `FastForward.lua` and `FastForward.json` into `%APPDATA%\Balatro\Mods\FastForward\`.
